# MSSQL

## For Windows: ##
1.  From cmd `git clone https://github.com/runlevel-4/mssql`
2.  `Windows\install_mssql.bat`


## For Linux: ##

### Ubuntu ###
1.  `git clone https://github.com/runlevel-4/mssql`
2.  `sh linux/install_sql_ubuntu_20_04.sh`

### CentOS 8 ###
1.  `git clone https://github.com/runlevel-4/mssql`
2.  `sh linux/install_sql_centos_8.sh`
